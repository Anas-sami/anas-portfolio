# Anas Sami - Personal Portfolio

A professional, handcrafted responsive portfolio website built from scratch using HTML, CSS, and modern JavaScript features. This site serves as an interactive hub showcasing my academic background, technical skills, and verified training credentials in Computer Science and Artificial Intelligence.

## 🛠️ Key Features

- **Modern Tech-Noir UI**: A premium dark-themed interface engineered manually without generic AI generators or rigid templates, ensuring a human-crafted look and feel.
- **Dynamic JavaScript Diagnostic Tool**: An embedded, interactive terminal emulator that allows users to run simulated system network diagnostics in real time.
- **Professional Accolades Showcase**: Dedicated credential modules highlighting verified certifications with absolute fidelity to the original issuers.
- **Native Navigation & Direct Redirection**: Standard-compliant, high-sillage routing built natively to bridge seamless redirects to LinkedIn, GitHub, and secure email client inboxes.
